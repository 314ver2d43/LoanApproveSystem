 # LoanApproveSystem

Credit approval data analysis


A data analysis project using Python and visualization libraries to investigate key factors influencing bank loan approvals

# About the project:


This project analyzes a dataset containing information on bank loan applicants. The objective is to identify patterns and determine which factors (such as income level and education) have the greatest impact on the bank's decision to approve or reject an application


# What was used:


    Python


    Pandas


    Matplotlib


    Scikit-learn (SimpleImputer)

    
  Income Analysis (ApplicantIncome):

    Analysis using a boxplot revealed notable outliers corresponding to rich clients with high incomes.

    The majority of applicants have a modest income (up to 20000$).


  Approval Status:

    Overall statistics show that loans are approved (Y) more frequently than they are rejected (N).


  Education Level:

    The client base consists primarily of people with higher education, whose numbers significantly exceed those of individuals without it


# Conclusion


Based by the visualizations, the typical applicant most likely to receive a loan is an educated and high-earning person.
